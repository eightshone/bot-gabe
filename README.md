# BOT GABE
A load testing cli tool that simulates bots behavior

# Quick start
## Usage
`⚠️ COMING SOON!`
## Development
### Clone project
`git clone git@github.com:eightshone/bot-gabe.git`

### Install dependencies
| Yarn           | NPM     |
| -------------- | ------- |
| `yarn install` | `npm i` |

### Start the project
| Yarn       | NPM           |
| ---------- | ------------- |
| `yarn dev` | `npm run dev` |
